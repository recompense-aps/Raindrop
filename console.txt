==========================================================
===         THE DEV CONSOLE                            ===
==========================================================

Press |ctrl + tab| to open the console
Press |enter| to execute the command

WARNING! if you enter badly formatted values (like text where
numbers should be) the game will probably crash

use like this:
set {category} {property} {value}

example:
set FreeFall ObstaclesPerLayer 20
